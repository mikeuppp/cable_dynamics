The Mooring Design and Dynamics Matlab package is software to help design and test single wire oceanographic moorings and towed bodies. There are several associated webpages that <a href="https://rkdewey.github.io/Mooring-Design-and-Dynamics/index.html">descibe the software</a>, a <a href="https://rkdewey.github.io/Mooring-Design-and-Dynamics/menu.html">download link page</a>, and a comprehensive <a href="https://rkdewey.github.io/Mooring-Design-and-Dynamics/mdd.html">Users Guide</a>.
